# Tian Qiu personal website (GitHub Pages)

## Quick start (GitHub Pages)
1. Create a repo named: <YOUR_GITHUB_USERNAME>.github.io
2. Upload the contents of this folder to the repo root (index.html, research.html, publications.html, about.html, assets/)
3. In GitHub: Settings → Pages → Deploy from branch → main / root
4. Your site will appear at: https://<YOUR_GITHUB_USERNAME>.github.io

## Customize
- Replace the Twitter/X link in about.html
- Add ORCID / LinkedIn / GitHub links if you want
- Optionally add a headshot image: assets/headshot.jpg and reference it on pages
